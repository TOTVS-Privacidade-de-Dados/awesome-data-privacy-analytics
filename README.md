# ***Data Privacy and Analytics Resources***

​                                                  ***"O conhecimento é como uma lâmpada acesa na escuridão da ignorância." - Confúcio***

Bem-vindo ao repositório GitHub "Privacidade de dados e recursos analíticos"! Este repositório é uma coleção com curadoria de leituras, ferramentas, estruturas e cursos on-line essenciais para indivíduos interessados em se aprofundar nos domínios da privacidade e análise de dados.

## **1. Become a Data Scientist (Udacity)**

O curso de [Data Scientist da Udacity](https://www.udacity.com/course/data-scientist-nanodegree--nd025) é opção **para quem quer atuar na prática de ciência de dados** e já entende conceitos como *Machine Learning* e já domina [Python](https://www.remessaonline.com.br/blog/python-descubra-quais-sao-as-suas-funcoes-na-programacao/). Também é importante ter skills de SQL, probabilidade, estatística e organização e visualização de dados (CSV e JSON).

A Udacity é uma instituição localizada no [Vale do Silício](https://www.remessaonline.com.br/blog/glossario/vale-do-silicio/) com parcerias de grandes nomes, como Google e Facebook e Amazon. O curso é dividido em módulos, com uma duração estimada de 4 meses. O aluno interessado pode comprar o curso e realizá-lo nesse período ou optar por pagar mensalmente, caso precise de mais tempo para a conclusão.

Ao longo das aulas, é possível trabalhar em projetos desenvolvidos por especialistas e contar com uma mentoria especializada. O curso está disponível apenas na língua inglesa.

Os principais tópicos do curso *Become a Data Scientist* da Udacity são:

- **Solving Data Science Problems:** resolução de problemas usando *Data Science*;
- **Software Engineering for Data Scientists:** engenharia de software para *Data Scientists*;
- **Data Engineering for Data Scientists:** engenharia de dados para *Data Scientists*;
- **Experiment Design and Recommendations:** design experimental e recomendações;
- **Data Science Projects:** projetos de *Data Science*.

## **2. Data Analyst (Udacity)**

## O curso de [Data Analyst](https://www.udacity.com/course/data-analyst-nanodegree--nd002) da Udacity é focado para quem deseja ser **analista de dados**. Aqui, é necessário conhecimento em linguagens de Python e SQL.

O analista de dados terá um aprendizado direcionado ao **gerenciamento e interpretação dos dados recebidos**. Ele é completamente online e tem 4 meses para sua conclusão. Além disso, conta com mentorias e as demais vantagens propostas pela instituição de ensino.

Porém, atenção: como os cursos da Udacity são desenvolvidos para um público internacional, eles são ministrados em inglês. Ainda não há legendas ou mesmo dublagens para as aulas. Caso você ainda não seja fluente no idioma, vale a pena conferir [cursos de inglês direcionados para programadores](https://www.remessaonline.com.br/blog/curso-de-ingles-para-programadores-conheca-5-opcoes/).

## **3. Applied Data Science with Python (Coursera)**

O curso de *Data Science* [Ciência de Dados Aplicada com Python](https://www.coursera.org/specializations/data-science-python?ranMID=40328&ranEAID=je6NUbpObpQ&ranSiteID=je6NUbpObpQ-ecQuzVDwzi1mSSZavb0Wxg&siteID=je6NUbpObpQ-ecQuzVDwzi1mSSZavb0Wxg&utm_content=10&utm_medium=partners&utm_source=linkshare&utm_campaign=je6NUbpObpQ), oferecido pela Coursera em parceria com a Universidade de Michigan, é um dos melhores da área. Ele é destinado para quem deseja ter um aprendizado mais completo e ainda não é especialista em Python. Ele cobre todo o assunto de *Data Science* relacionado a Python.

O curso é composto por 5 outros da plataforma. Isso requer um aprendizado básico na temática abordada por parte do aluno.

Os cursos integrados do curso Ciência de Dados Aplicada com Python são:

- Introdução ao *Data Science* em Python;
- Gráficos e Representação de Dados em Python;
- Machine Learning em Python;
- Mineração de Texto Aplicada em Python;
- Análise de Rede Social Aplicada em Python.

O curso pode ser concluído em 4 meses, caso o aluno cumpra uma carga horária de 10 horas semanais. A plataforma requer uma assinatura para acessar todos os materiais. Contudo, também disponibiliza auxílio financeiro para alunos que não têm como arcar com os valores. O curso é em inglês, mas legendas são disponibilizadas em outros idiomas.

## **4. Data Science Fundamentals with Python and SQL (Coursera)**

O curso [Data Science Fundamentals with Python and SQL](https://www.coursera.org/specializations/data-science-fundamentals-python-sql?ranMID=40328&ranEAID=je6NUbpObpQ&ranSiteID=je6NUbpObpQ-_vl4qA1qOlqsV.kdIgjinA&siteID=je6NUbpObpQ-_vl4qA1qOlqsV.kdIgjinA&utm_content=10&utm_medium=partners&utm_source=linkshare&utm_campaign=je6NUbpObpQ), oferecido pela IBM na [Coursera](https://www.coursera.org/), é direcionado para quem está **iniciando na área**. Ou seja, trata-se de um aprendizado mais simples, com um período menor de duração e conceitos mais básicos.

Ainda assim, esse é um dos melhores cursos de *Data Science*, por oferecer certificados reconhecidos e possibilitar ao aluno uma base sólida no assunto.

Cursos envolvidos na formação do curso *Data Science Fundamentals with Python and SQL*:

- Ferramentas para *Data Science*;
- Python para *Data Science*, Inteligência Artificial e Desenvolvimento;
- Projeto Python para *Data Science*;
- Estatística para *Data Science* com Python;
- Banco de Dados para *Data Science* com Python.

A formação dura cerca de 2 meses com uma dedicação semanal de 10 horas. A programação requer prática e memorização. Então, manter uma rotina fixa de estudos é fundamental.

## **5. Data Scientist with Python (DataCamp)**

O curso [Data Scientist with Python](https://www.datacamp.com/tracks/data-scientist-with-python?irclickid=Wz01siSYpxyLUr6TAxQoNQRiUkBR8wUQNX3eUo0&irgwc=1) **não tem pré-requisitos para quem deseja aprender \*Data Science\***. Ele é disponibilizado pelo [DataCamp](https://www.datacamp.com/) e conta com alunos de diversas instituições como Google, [PayPal](https://www.remessaonline.com.br/blog/comprar-com-paypal/), [Ebay](https://www.remessaonline.com.br/blog/ebay-2/), Uber etc. Ao todo, são 90 horas de aulas nas quais exercícios práticos são aplicados para melhor fixação do conteúdo, além da realização de diversos projetos. 

Vale ressaltar que ele reúne cerca de 25 módulos de conhecimento, como exclusão, importação e programação de dados no Python. O curso é em inglês e pode ser contratado tanto em plano individual quanto em plano coletivo — ideal para quem quer economizar e estudar com colegas.

## **6. Data Analysis in Excel** **(DataCamp)**

O curso de [Data Analysis in Excel](https://www.datacamp.com/courses/data-analysis-in-excel) é em inglês e um de seus 3 módulos é gratuito. Nele você vai **conhecer e praticar as principais funções do Excel no uso de dados**. O curso ajuda a usar o Excel para analisar e pesquisar dados. Ele dura apenas 4 horas e dispõe de quase 50 módulos só de exercícios.

Além disso, você acompanha 12 videoaulas que abordam preparação e exportação de dados para outros sistemas e plataformas comuns nas organizações. Sempre com a integração do seu trabalho com outras atividades do setor de TI em mente.

## **7. Data Science: Visualization (edX)**

O curso de [Data Science: Visualization](https://www.edx.org/course/data-science-visualization) da [edX](https://www.edx.org/) é uma preparação de **nível introdutório** em inglês muito valioso no mercado. Ele dura 8 semanas e é ensinado pelo professor de bioestatística em [Harvard](https://www.remessaonline.com.br/blog/harvard/), Rafael Irrizarry. 

Os conhecimentos adquiridos envolvem visualizar e explorar a interpretação de dados, considerando os preconceitos e outros erros. É uma excelente oportunidade para quem deseja iniciar na área e precisa contar com um certificado de instituição de renome.

O curso ensina a utilizar o ggplot2 para a criação de gráficos personalizados, além de mostrar erros comuns das visualizações em gráficos. Além disso, dá uma visão mais analítica e precisa diante dos dados coletados.

## **8.** **Data Privacy and Technology (Harvard)**

O [Data Privacy and Technology](https://pll.harvard.edu/course/data-privacy-and-technology?delta=3) é um curso de *Data Science* que **explora os avanços tecnológicos na área** e debate sobre a coleta e uso de dados de terceiros. Ele também questiona até que ponto as políticas de privacidade e acesso anônimo realmente protegem os usuários.

A que ponto as empresas podem ser responsabilizadas legalmente por vazamentos e uso malicioso de informações pessoais de clientes? Esse é o tipo de conteúdo que ajuda você a desenvolver o seu senso crítico. Além disso, permite conhecer mais sobre legislação cibernética mundial e a se aprofundar no uso de dados no mercado.

A inscrição no curso é paga e garante o acesso ao material didático, às aulas e à certificação. Ele é disponibilizado por [Harvard](https://pll.harvard.edu/subject/data-science), dura 5 semanas e é ministrado em inglês.

## **9. Power BI Expert na Prática (Viscari)**

O curso [Power BI Expert na Prática](https://www.viscari.com.br/power-bi-expert?ref=D70037426P), da [Viscari](https://www.viscari.com.br/), vai te **introduzir para a área de análise de dados**. Com a ferramenta [Power BI](https://www.remessaonline.com.br/blog/power-bi-conheca-a-ferramenta-como-utiliza-la-e-vantagens/) você ficará apto a transformar banco de dados em relatórios dinâmicos e formular *dashboards.*

O objetivo desse curso é mostrar na prática do básico até conhecimentos avançados. Você aprenderá a criar relatórios funcionais, dinâmicos e com visuais que prendem atenção.

A carga horária é de 30 horas, sendo 11 módulos e acesso vitalício.

## **11. Power BI Completo – Do Básico ao Avançado (Udemy)**

O curso de *Data Science* [Power BI Completo – Do Básico ao Avançado](https://www.udemy.com/course/power-bi-completo-do-basico-ao-avancado/?ranMID=39197&ranEAID=OjkYjeK5T1I&ranSiteID=OjkYjeK5T1I-6hOFoMxw289n6bUREl1YNw&utm_source=aff-campaign&utm_medium=udemyads&LSNPUBID=OjkYjeK5T1I) da Udemy consiste em aulas **passo a passo** sobre tudo o que você precisa saber sobre **Power BI**.

O curso é 100% prático e conta com 83 aulas, com duração de 10 horas. O tempo todo o aluno estará aplicando e exercitando, para se preparar da melhor forma possível a qualquer caso que tenha que resolver na vida profissional.

## **12. Curso de Análise de Dados e Big Data – Excel (Udemy)**

O curso de [Análise de Dados e Big Data – Excel da Udemy](https://www.udemy.com/course/probabilidade-estatistica-excel/?ranMID=39197&ranEAID=OjkYjeK5T1I&ranSiteID=OjkYjeK5T1I-DFM.sRyYVlCpmJQjPHxOng&LSNPUBID=OjkYjeK5T1I&utm_source=aff-campaign&utm_medium=udemyads) traz uma **abordagem prática sobre aplicações de ferramentas** do cientista de dados. As funcionalidades estudadas são de probabilidade, estatística e análise de dados, a fim de solucionar problemas reais de diversas áreas de atuação profissional.

A proposta é que o profissional consiga tomar as melhores decisões sobre fenômenos que envolvem incerteza. Você aprenderá a criar histogramas, boxplots, calcular medidas, resumo, identificar funções e densidade de probabilidade para conjuntos de dados usando suplementos do Excel.

Não é necessário ter conhecimento avançado em Excel, mas sim uma familiaridade com o programa. O curso tem uma carga horária de 12,5 horas.

## **13. Arquitetura de Software, Ciência de Dados e Cybersecurity (PUCPR)**

O curso de [Arquitetura de Software, Ciência de Dados e Cybersecurity da PUCPR](https://posdigital.pucpr.br/cursos/ciencia-de-dados-arquitetura-de-software-e-cybersecurity) oferece aos alunos um **desenvolvimento das habilidades** em construir sistemas, escolher componentes, direcionar estratégias, fazer migrações e criar ambientes seguros.

## **14. Formação Data Science (Alura)**

O curso de [Formação Data Science da Alura](https://www.alura.com.br/formacao-data-science) ensina técnicas de análise, estatística, modelagem e visualização de dados. Os alunos também terão acesso a 3 podcasts sobre a temática para audição.

A metodologia do curso engloba conhecimentos de Python, bibliotecas e ferramentas paralelas focadas em linguagem de programação.

As aulas ensinarão a coletar, preparar e explorar os dados, além de transmitir informações por meio de gráficos. O curso de Formação *Data Science* da Alura tem uma carga horária de 102 horas, e um total de 5 professores envolvidos.
